# Feature: tool-schema

## Overview

Implement contract-to-MCP tool schema mapping. Transforms ContractedAPI contracts into MCP tool definitions.

## Key Deliverables

- Contract to MCP tool mapping
- Tool name derivation
- Input schema from request schema
- Path parameter as required inputs

## Acceptance Criteria

- [ ] Tools generated from contracts
- [ ] Input schemas match requests
- [ ] Path params included
- [ ] Descriptions preserved
