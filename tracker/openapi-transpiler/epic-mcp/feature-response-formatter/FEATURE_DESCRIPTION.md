# Feature: response-formatter

## Overview

Implement LLM-friendly response formatting for MCP tool results.

## Key Deliverables

- JSON to text transformation
- Array formatting
- Object formatting
- Error formatting
- Large response truncation

## Acceptance Criteria

- [ ] Responses readable by LLM
- [ ] Arrays listed nicely
- [ ] Objects formatted as key-value
- [ ] Errors are helpful
