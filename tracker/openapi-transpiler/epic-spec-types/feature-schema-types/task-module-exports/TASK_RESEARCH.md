# Task: module-exports - Research

## Deno Module Convention

In Deno, `mod.ts` is the conventional entry point for modules (equivalent to `index.ts` in Node.js).

## Export Organization

Group exports logically:
1. Core JSON types first
2. Schema inference types second
3. Utility types last (if any)
