# Feature: cli-entry

## Overview

Implement the main CLI entry point with global options, configuration loading, and command structure initialization.

## Key Deliverables

- Main `contracted` command entry point
- Global options (--base-url, --format, --verbose)
- Configuration file loading
- Environment variable support
- Version and help commands

## Acceptance Criteria

- [ ] CLI starts and parses arguments
- [ ] Global options work correctly
- [ ] Config file loads if present
- [ ] Environment variables override config
