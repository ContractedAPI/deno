# Feature: command-tree

## Overview

Implement command tree generation from contract paths. Transforms API paths into hierarchical CLI commands.

## Key Deliverables

- Path to command tree conversion
- Path parameters as positional arguments
- Method to subcommand mapping
- Command aliasing support

## Acceptance Criteria

- [ ] Paths become command groups
- [ ] Path params become positional args
- [ ] Methods map to subcommands
