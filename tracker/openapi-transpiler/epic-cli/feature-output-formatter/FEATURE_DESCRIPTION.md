# Feature: output-formatter

## Overview

Implement output formatting for CLI responses (JSON, table, plain text).

## Key Deliverables

- JSON formatter (pretty print)
- Table formatter for arrays
- Plain text formatter
- Color support with NO_COLOR respect

## Acceptance Criteria

- [ ] JSON output works
- [ ] Table output renders correctly
- [ ] Plain output for piping
- [ ] Colors can be disabled
